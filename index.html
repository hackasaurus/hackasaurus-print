Tips

  • It's possible to remove Firefox's default headers and footers from the page layout; see the Firefox Support article entitled Printing A Web Page for instructions.
  • The page design was created with the Failbook hack as a test case, so try to make your hack have a similar aspect ratio, or else some information might not fit on the first printed page.
  • If there's anything on the printed page that you want to change, such as the title, description, or dimensions of the screenshot, you can use the Hacker Goggles to do so. Any changes you make with them will show up in the printed version.
